# NeoComics

NeoComics é um e-commerce de quadrinhos da Marvel, onde você pode navegar pela listagem de produtos, ver detalhes de cada quadrinho, fazer buscas e adicionar itens ao carrinho.

## Tecnologias usadas

- Next.js  
- React  
- Redux Toolkit  
- Styled Components
